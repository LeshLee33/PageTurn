# GIT

пул

```
git branch -M master
git pull
```

пуш

```
git add .
git commit -m "название коммита"
git branch -M master
git push -u origin dev
```

объединение

```
git checkout master
git merge dev
```

---

# FRONT-END

установите node js

для запуска проекта:
```
cd src/frontend
npm run start
```

# BACK-END

для получения библиотек из файла requirements.txt нужно прописать в консоль следующие команды

```
cd src/server
pip install -r requirements.txt

```
